分支结构 注意: 1. 括号闭合要正确 2. 尽量保持代码的缩进关系 3. 所有的符号都是英文输入法 -> 小括号 () -> 大括号 {} if 结构 语法： if (条件) { 条件为 true 的时候执行 } else { 条件为 false 的时候执行 } + 代码执行顺序的一种 + 我们的代码应该是 从上到下 依次执行 + 分支结构, 就是到了某一个位置的时候, 选择性的执行哪一行代码 + 当代码到达分支位置的时候 + 根据条件决定执行 a 行 还是 b 行 + 分支结构的代码有两种语法 1. if 语句 2. switch 语句 分支结构 - if 语句 + 如果 ... 就 ... + if 语句的语法分成四种形式 1. if (条件) { 要执行的代码 } -> 只要条件成立(条件结果为 true), 那么就执行后面 {} 里面的代码 -> 如果条件不成了(条件结果为 false), 那么就不执行后面 {} 里面的代码 2. if (条件) { 条件成立就执行 } else { 条件不成立就执行 } -> 只要条件成立, 那么就在执行 if 后面的 { } 里面的代码 -> 只要条件不成立, 那么就执行 else 后面的 { } 里面的代码 -> 我们这两个 { }, 能且只能执行一个 3. if (条件一) { 条件一成立就执行 } else if (条件二) { 条件二成立就执行 } ... -> 哪一个条件成立, 就执行哪一个 if 后面的 { } 里面的代码 -> 如果前面的条件成立了, 后面的就不在判断了, 直接忽略 -> 如果所有的条件都不成立, 那么就一个都不执行 4. if (条件一) { 条件一成立就执行 } else if (条件二) { 条件二成立就执行 } else { 所有条件都不成立的时候执行 } -> 哪一个条件成立, 就执行哪一个 if 后面的 { } 里面的代码 -> 如果前面的条件成立了, 那么后面的就不要判断了, 直接忽略 -> 如果所有条件都不成立, 那么就执行 else 后面的 { } 里面的代码 -> 所有的 {}, 能且只能执行一个 switch 分支语句： 分支结构 - switch 语句 + 也是分支结构的一种, 用来做条件选择 + switch: 道岔 + 语法: switch (你要判断的值) { 一条一条的路 case 情况一: 你要判断的值满足情况一的时候要执行的代码 break case 情况二: 你要判断的值满足情况二的时候要执行的代码 break default: 上面所有的 case 都不满足的时候会执行这里的代码 + 注意: 1. 小括号里面不要符号, 就是直接写变量 2. case 后面先写要判断的内容, 再写一个冒号 3. 每一个 case 后面要记得写 break 4. 你要判断的值(也就是你写在小括号里面的内容) -> 和你的 case 后面的内容必须是 === 的情况才可以 -> 如果只是 == 的情况, 是不可以的 5. switch 只能判断一个变量是不是 全等于 一个 case -> 不能判断一个范围, 大于 小于 之类的都不可以 6. default 可以写可以不写 -> 你写了, 就是在所有的 case 都不满足的时候有代码执行 -> 你不写, 就是在所有的 case 都不满足的时候没有代码执行 switch 的 case 穿透 + 当你的某一个 case 后面没有写 break 的时候 + 会进行一个 case 穿透的效果 -> 也就是执行完毕当前的 case 以后, 会继续执行下一个 case -> 不管下一个 case 的条件是不是满足, 都会向下执行 -> 直到一个 break 为止 + 在穿透的时候, 不是每一次都从第一个开始穿透 -> 而是从满足条件的第一个开始向下穿透 和 if 语句的区别 1. 语法上的区别 2. 判断内容的区别 -> if 可以进行范围判断 -> switch 只能进行准确固定值的判断 基础写法： switch 的穿透：
