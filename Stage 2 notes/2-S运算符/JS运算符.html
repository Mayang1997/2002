运算顺序是从左至右的,当 且 和 或 运算符都在都情况下,执行的是先且后或 JS运算符：数学运算符 一：基础运算符 有5个 +号运算只有两个意义 1：拼接，运算符任意一边有字符串的时候 2：数学运算加法：运算符两边都是数字或者布尔时 true值为1 false为0 - 号运算： 1：字符串也能运算减法，有结果为合法数字，否结果为NaN * 运算： 1：字符串也能运算*法，有结果为合法数字，否结果为NaN / 法运算 1：字符串也能运算/法，有结果为合法数字，否结果为NaN % 取余(摩) ： 1：字符串也能运算%法，有结果为合法数字，否结果为NaN 取余就是除不尽的值。 被除数 / 除数 = 商 ... 余数（余数就是取余的值） 二：赋值运算符： 有6个 = 赋值 ： 并不是等于 += 加等于： 加号于等号的结合 a += b 等价于 a = a + b 1：规则与数学运算符的 + 号运算规则一样 -+ 减等于： 减号于等号的结合 a -= b 等价于 a = a - b 1：规则与数学运算符的 - 号运算规则一样 *= 乘等于： 乘号和等号的结合 a *= b 等价于 a = a * b 1：规则与数学运算符的 - 号运算规则一样 %= 摩等于： 取余和等号的结合 a *= b 等价于 a = a * b 1：规则与数学运算符的 % 号运算规则一样 三：比较运算符： 把两边的数据进行计算 得结果 比较运算法得结果只有两种：true 和 false > 数值： 左大于右就是 true 右大于左就是false < 正常得数值对比 >= 大于等于 <= 小于等于 == 等于 1：不考虑数据类型 ，只有两边值一样，就是true 不一样就是false === 全等于 1：只有数值类型和数值都一样，才是true 只要数据类型和值有任意一个不一样, 就是 false !== 不全等 1：当两部数据类型不一样的时候, 直接就是 true 只有两边数据类型一样的时候, 才会去比较 值 如果值也是一样的, 那么就是 false 如果值不一样, 就是 true 三：逻辑运算符：数据与数据之间出现了逻辑关系 && 与(且)： 只有运算符两边都是 true 的时候, 才会得到 true 只要有任意一边是 false, 那么结果就是 false || 或 只要有任意一边是 true, 那么得到的结果就是 true 只有两边都是 false 的时候, 才会得到 false ! 非(取反) 如果你本身是一个 true, 那么结果就是 false 如果你本身是一个 false, 那么结果就是 true 注意: 当你取反一个运算结果的时候, 需要小括号 !a + b, 先计算 !a 的结果, 用结果和 b 进行加法运算 !(a + b), 先计算 a + b 的结果, 把结果取反 四：自增自减运算符：重要 ++加加运算符 作用：让本身得值加1 --减减运算符 作用：让本身得值减1 加加运算符 + 有两种使用方式 1. 前置++ => 是把符号写在变量的前面 => 只要这个代码一执行, 就会改变自身变量的值 => 当前置 ++ 参与运算的时候, 会先改变自身的值 -> 用改变后的值去参与运算 2. 后置++ => 是把符号写在变量的后面 => 只要这个代码一执行, 就会改变自身变量的值 => 当后置 ++ 参与运算的时候, 会先把变量的值拿来参与运算 -> 然后再改变变量的值 减减运算付符 + 有两种使用方式 1. 前置-- => 把符号写在变量的前面 => 只要这个代码一执行, 就会改变自身的值 => 当前置-- 参与运算的时候, 会先改变自身的值 -> 用改变后的值去参与运算 2. 后置-- => 把符号写在变量的后面 => 只要这个代码一执行, 就会改变自身的值 => 当后置 -- 参与运算的时候, 会先把变量的值拿到参与运算 -> 然后再改变变量的值
